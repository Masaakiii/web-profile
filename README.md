## 🚀 GitHub Pages デプロイ手順（備忘録）

1. GitHub にリポジトリを作成（例：web-profile）  
2. 以下のファイルをアップロード：
   - `index.html`
   - `style.css`
   - 画像ファイル（例：`KiKisama.png`）  
3. **GitHub Pages の設定：**  
   `Settings` → `Pages` → `Source` を `main` + `/ (root)` に設定  
4. 数秒後に公開 URL が生成される  
   - URL形式：`https://ユーザー名.github.io/リポジトリ名`
